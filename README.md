# Gym Management System (Python OOP)

This project was developed to practice and reinforce **Object-Oriented Programming (OOP)** principles. It is a gym membership and payment management system designed to demonstrate core OOP concepts through a real-world application.

## Features

- **Member Management:** Register new members, delete existing members, and search members by their ID.
- **Automatic Date Calculation:** Membership expiration dates are automatically calculated based on the selected package (1, 3, 6, or 12 months) using Python's `datetime` module.
- **Cash Register Report:** Logs all payment transactions and provides real-time revenue tracking.
- **Membership Renewal:** Extend or renew existing memberships with flexible subscription periods.
- **Encapsulation:** Member data is protected using encapsulation, with controlled access through getter and setter methods.
- **Composition:** The `Gym` class contains `Member` and `Payment` objects, demonstrating composition and hierarchical object relationships.
- **Input Validation:** Custom validation for phone numbers and email addresses to ensure valid user input.

## Tech Stack

- **Language:** Python
- **Programming Paradigm:** Object-Oriented Programming (OOP)
- **Modules:** `datetime`
- **Storage:** File I/O (file-based data storage)
